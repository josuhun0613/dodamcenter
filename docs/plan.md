# 도담센터 웹사이트 기획서

> 이 문서는 대화를 통해 지속적으로 업데이트됩니다.
> 최종 수정: 2026-03-02

## 1. 프로젝트 개요

| 항목 | 내용 |
|------|------|
| 프로젝트명 | 도담센터 웹사이트 |
| 성격 | 심리상담 전문센터 홍보 + 예약 사이트 |
| 타겟 고객 | 심리상담이 필요한 성인 |
| 디자인 톤 | 깔끔하고 전문적인 (미니멀, 신뢰감) |
| 배포 | Vercel |
| DB | Firebase (전환 예정, 현재 로컬 JSON 파일) |

## 3. 남은 핵심 작업

### 우선순위 높음 (Must Have)
- [ ] Firebase 연동 (현재 로컬 JSON → Firebase Firestore로 전환)
- [ ] 관리자 인증 보안 강화 (하드코딩 → Firebase Auth)
- [ ] 관리자 페이지 라우트 보호 (미인증 접근 차단)
- [ ] 상담사 관리 CRUD 완성 (추가/수정/삭제)

### 우선순위 중간 (Should Have)
- [ ] 문의/예약 알림 시스템 (이메일 또는 카카오 알림)
- [ ] 관리자 전화예약 관리 페이지
- [ ] 프로그램 데이터 동적 관리 (현재 하드코딩)
- [ ] 이미지 업로드 기능 (상담사 사진, 리뷰 이미지)
- [ ] 환경변수 정리 (.env.local 구성)

### 우선순위 낮음 (Nice to Have)
- [ ] SEO 최적화 (sitemap 동적 생성, 메타태그 보강)
- [ ] 성능 최적화 (이미지 최적화, 코드 스플리팅)
- [ ] 접근성(a11y) 개선
- [ ] 관리자 대시보드 분석 기능 강화

## 5. 디자인 가이드 요약

- **컬러**: 베이지(#FBF9F7) + 브라운 액센트(#8B7355) + 블랙(#1A1A1A)
- **폰트**: Pretendard(본문) + Playfair Display(장식)
- **모서리**: 카드 `rounded-2xl`, 버튼 `rounded-full`
- **애니메이션**: Framer Motion 스크롤 트리거, 0.3~0.6s duration
- **레이아웃**: 모바일 퍼스트, `max-w-7xl` 컨테이너
